# J'aime ton impro

Une application permettant aux membres d'un groupe d'improvisation
de laisser des appréciations anonymes (ou privées) aux autres membres,
et d'afficher un message "mis en avant" par chaque membre.

## Stack

- Frontend : React + Vite + TailwindCSS
- Backend : .NET 7+ (à venir)
- Hosting : Raspberry Pi (Docker + Traefik/npm)

## Développement local

### Frontend

```bash
cd frontend
npm install
npm run dev
